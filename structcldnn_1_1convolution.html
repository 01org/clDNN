Detailed Description

Performs forward spatial convolution with weight sharing. Also supports built-in Relu cldnn_activation_desc available by setting it in arguments.

Parameters are defined in context of "direct" convolution, but actual algorithm is not implied. Look into docs/size_offset_stride_padding.html for description how size, offsets, stride & padding parameters work.
