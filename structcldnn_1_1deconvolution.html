Detailed Description

Performs transposed convolution. Also supports built-in Relu activation available by setting it in arguments.

Deconvolution is similar to convolution layer with the weights flipped on the axis and stride and input padding parameters used in opposite sense as in convolution. Look into docs/size_offset_stride_padding.html for description how size, offsets, stride & padding parameter work.
